# Contact Manager in C

Sistema simples de agenda de contatos desenvolvido em C.

## Funcionalidades
- Adicionar contatos
- Listar contatos
- Buscar contato por nome

## Conceitos aplicados
- Structs
- Manipulação de strings
- Estruturas de controle

## Biblioteca utilizada

Este projeto utiliza a biblioteca "pss.h" para entrada de dados no terminal.

Disponível em:
https://github.com/Prof-Saulo_Santos/pss-library

#### Observações

- O arquivo "pss.h" não está incluído no repositório e deve ser adicionado manualmente.
- Os dados são armazenados apenas em memória (não persistem após o encerramento do programa).
